Grid repeat demo

Intro

This demo shows the effect of using the repeat(repetitions, track list fragment) function as track list for the columns of a grid.

The 200px track list fragment creates fixed 200px columns, the minmax(200px, 1fr) creates flexible 200px columns that grow if space is left over. Vary the viewport width to see the effect of the flexible columns.

There are three items that are auto placed in the first three columns, any further columns stay empty.

The fixed repetition creates exactly five 200px columns. The auto-fill repetition creates as many 200px columns as fit. The auto-fit repetition is similar to auto-fill, except it doesn't create empty columns.

Note: The viewport width must be at least 800px + 3rem wide to see the difference between auto-fit and auto-fill repetitions and at least 1000px + 4rem wide to see the difference between fixed and flexible track sizes for a fixed amount of repetitions.
