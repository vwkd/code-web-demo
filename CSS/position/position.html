Position demo

Intro

This demo shows the different position values applied to the red box. It's parent is colored blue and the parent siblings are colored grey.

The box has a offset of top: 50px which has different effects based on the value of the position property. For clarity no horizontal offset is applied. It would have analogous effects in the horizontal axis depending on overflow.

In the static, relative and sticky case the parent is the containing block, since it's the nearest ancestor box with ODT not inline. The parent has position: relative to be the containing block for the absolute case. In the fixed case the viewport is the (initial) containing block. Also in the sticky case the box has the <html> element as nearest scrollport.
