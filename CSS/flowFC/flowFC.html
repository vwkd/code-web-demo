Intro

Given the following markup you can toggle the following properties. 


  <span class="parent-sibling">a</span>
  <span class="parent">
      <span>ba</span>
      <span class="child">bb</span>
      <span>bc</span>
  </span>
  <span class="parent-sibling">c</span>

Parent siblings get parent ODT and IDT fixed to flow. Child siblings default to inline flow.

Note: Check browser compatibility first for multi-valued display type on caniuse or MDN.
